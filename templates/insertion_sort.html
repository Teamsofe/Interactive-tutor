Ever arranged playing cards in your hand? You pick a card and insert it into the right place among the already sorted ones. That‚Äôs exactly how Insertion Sort works!

It builds the final sorted array one item at a time by comparing each new element with those before it and inserting it into the correct position.

üìù Pseudocode

For i from 1 to n-1:
    key = A[i]
    j = i - 1
    While j >= 0 and A[j] > key:
        A[j + 1] = A[j]
        j = j - 1
    A[j + 1] = key
üß† Line-by-Line Explanation:
  • We assume the first item is sorted.
  • Pick the next item and compare it with the ones before it.
  • If it's smaller, shift larger items one position to the right.
  • Insert the picked item into the correct position.
üìå More Insights
  • Why "Insertion"? Elements are inserted in the correct spot as we move forward.
  • Efficiency: Time complexity is O(n¬≤) ‚Äì like Bubble Sort ‚Äì but often performs better in practice.
  • Best Use: Small or nearly sorted datasets.
