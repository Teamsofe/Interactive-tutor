üß© Understanding Linear Search (The Friendly Way)

Imagine looking for your favorite book on a shelf, one by one. You check each book until you find it. This is exactly how Linear Search works!

It checks each element in a list until the target is found or all elements have been searched.

üìù Pseudocode

For i from 0 to n-1:
    If A[i] == target:
        Return i
Return "Not Found"
üß† Line-by-Line Explanation:
  • We start from the beginning of the list.
  • Check each item to see if it matches the target.
  • If a match is found, return the index.
  • If no match is found after checking all items, return "Not Found".
üìå More Insights
  • Why "Linear"? It goes through the list in a linear fashion, one element at a time.
  • Efficiency: Time complexity is O(n), as it checks every element.
  • Best Use: Small lists or unsorted data.
